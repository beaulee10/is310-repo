# NBA Championship Maze

By Beau Lee

## Overview

You've stepped onto the hardwood. The lights are bright, the crowd is roaring, and the championship trophy is hidden somewhere in the league. Paths twist through famous arenas, and not every locker room holds a prize.

One clue will lead you to the victory; the rest are just benchwarmers. Can you find the trophy?

## Necessary Commands

| Command | Purpose |
| :--- | :--- |
| `ls` | List directory contents |
| `cd` | Move between directories |
| `cat` | Read a text file |
| `ls -a` | Show hidden files and folders |
| `cd ..` | Go back one directory |
